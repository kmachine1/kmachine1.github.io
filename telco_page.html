Telco Customer Churn Prediction

This is a Machine Learning project which is done using Python Packages. The aim is to accuractely predict which customers will leave and which customers will stay.

The loading and formatting of the data is done mostly with the pandas packages. Then other packages are used for data transformation and resampling.

The goal is to choose and tune the best model after formatting the data in the best shape, such that there is no underfitting or overfitting and the model score is over 0.75.
